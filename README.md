# Where Does It Hurt?

An interactive, accessible Progressive Web App to help you understand your symptoms and identify possible pain sources.

## Features

- 📱 **Installable App**: Install on any device (mobile, tablet, desktop)
- 🔌 **Offline Support**: Works without internet connection
- ♿ **Fully Accessible**: WCAG 2.1 Level AA compliant
- ⏰ **Live Clock**: Real-time clock display in draggable window
- 🔗 **Quick Links**: Easy access to medical resources
- 🎨 **Beautiful UI**: Responsive design with smooth animations

## Installation

### Web Browser (Desktop/Mobile)
1. Open the app in your browser
2. Look for the "Install" prompt (varies by browser)
3. Click to install, or add to home screen

### Desktop (Chrome/Edge)
- Look for the install icon in the address bar
- Or: Menu → "Install app"

### Mobile (iOS)
- Tap Share → "Add to Home Screen"

### Mobile (Android)
- Tap Menu → "Install app"
- Or look for the install prompt

## Progressive Web App (PWA) Features

This app is a Progressive Web App with the following capabilities:

- **Service Worker**: Enables offline functionality
- **Web Manifest**: Allows app installation and configuration
- **Responsive Design**: Works on all device sizes
- **Background Sync**: Ready for future offline features
- **Installable**: Add to home screen or app menu

## Files

- `index.html` - Main page
- `manifest.json` - PWA manifest with app metadata
- `service-worker.js` - Service worker for offline support
- `about.html` - About page
- `diagnosis.html` - Diagnosis page
- `contact.html` - Contact page

## Accessibility

- Full keyboard navigation
- Screen reader support
- Focus indicators
- ARIA labels and live regions
- High contrast text

## Offline Usage

The app caches the main pages so you can:
- Access the home page without internet
- Read the about section
- Use the clock and draggable windows
- View previously loaded images

## Getting Started

1. Open `index.html` in a web browser
2. You'll see a loading screen (2.5 seconds)
3. The app will load with the main interface
4. Install as an app for the full native experience

## Technical Stack

- HTML5 with semantic markup
- CSS3 with responsive design

- Service Workers for offline support
- Web App Manifest for PWA features
